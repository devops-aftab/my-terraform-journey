# Lab 08: AWS ECS Deployment Strategies (EC2 vs. Fargate)

> **Flagship Project:** This lab evolved into a full production-grade reference architecture and now lives in its own dedicated repository!

### Project Highlights
* **EC2-Backed vs. Fargate (Serverless):** Side-by-side comparison of compute strategies.
* **Dynamic Host Port Mapping:** Utilizing Docker `bridge` mode on EC2 host fleets.
* **Production Networking:** Multi-AZ VPC, Application Load Balancers, and target group health tracking.
